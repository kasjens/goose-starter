# goose-starter

A beginner-friendly starter repo for getting up and running with AI using
[Goose](https://goose-docs.ai/) — an open-source AI agent you run on your own
machine.

If you're new to Goose, start here and work through the docs in order.

## Getting started

1. **[Install Goose](docs/getting-started.md)** — download Goose and set up a
   package manager (Scoop on Windows, Homebrew on macOS) so tools and
   dependencies install cleanly.
2. **[Import skills](docs/importing-skills.md)** — extend Goose with ready-made
   skills from public repositories, installed for you on request.
3. **[Use a GitHub Enterprise Copilot seat](docs/goose-github-enterprise-copilot.md)**
   *(optional)* — point Goose at your company's GitHub Enterprise Copilot
   instead of public `github.com`.

## About Goose

Goose is an AI agent that runs locally and can use tools, run commands, and
extend itself with skills. Learn more at the official docs:
<https://goose-docs.ai/>.
